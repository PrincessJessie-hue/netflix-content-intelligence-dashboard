# 🎬 Netflix Content Intelligence Dashboard


An end-to-end data analytics project exploring Netflix's global content strategy using Microsoft Excel and Microsoft Power BI.

## Project Information

**Role:** Data Analyst

**Tools:** Microsoft Excel, Microsoft Power BI, GitHub

**Project Type:** Exploratory Data Analysis (EDA) & Interactive Dashboard

**Dataset Size:** 7,787 Netflix titles

---

## Overview

## Overview

Netflix's content library has grown rapidly over the last decade, making it increasingly difficult to identify trends through manual analysis. This project explores Netflix's catalogue using Microsoft Excel and Power BI to uncover insights into content growth, regional production, genre distribution, audience targeting, and the balance between Movies and TV Shows.

The final deliverable is an interactive Power BI dashboard supported by exploratory data analysis and business recommendations.

---

## Business Questions

- How has Netflix's catalogue changed over time?
- Which countries contribute the most content?
- Which genres dominate the catalogue?
- Are Movies or TV Shows more common?
- Is Netflix primarily targeting adult audiences?

---

## Tools

- Microsoft Excel
- Microsoft Power BI
- GitHub

---

## Key Findings

- Netflix's catalogue expanded rapidly after 2015, peaking at 1,121 releases in 2018.
- The United States contributes the largest share of content, followed by India, the United Kingdom, Japan, and South Korea.
- Movies account for approximately 69% of all titles.
- Netflix primarily targets teen and adult audiences.
- The platform maintains a diverse catalogue across multiple genres.
